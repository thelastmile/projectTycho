About Us

We are current students in the San Quentin Code.7370 web development training class. During the "Project Phase" of the curriculum we were asked to collaborate with Princeton University's Dr. Michael Martinez-Bakker, an infectious disease ecologist, to create a data visual and interactive format to implement into Project Tycho website. We were given six weeks to code and design this project to be presented for actual use at Project Tycho. We are proficient in our understanding of HTML5, CSS3 and JavaScript, as well as in the additional JavaScript libraries of JQuery and D3. We applied our functional understanding these libraries to successfully complete this project that is now open for interaction. It was a very trying and educational experience but, also fun as we continue in our growth in writing code and in our overall web development education. Below, please take a moment to learn just a little more about us.

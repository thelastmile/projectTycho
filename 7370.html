ABOUT US

We are current students in the San Quentin Code.7370 web development training class. During the "Project Phase" of the curriculum we were asked to collaborate with Princeton University's Dr. Micaela Martinez-Bakker, an infectious disease ecologist, to implement an interactive data visualization for Project Tycho. We were given six weeks to code and design this project to be presented for actual use at Project Tycho. We are proficient in HTML5, CSS3 and JavaScript, as well as the JavaScript libraries JQuery and D3. We applied our functional understanding of these libraries to build this app. It was a very trying yet educational experience, but also fun as we continue our growth in writing code. Please take a moment to learn just a little more about us.
